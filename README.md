# 📊 Data Management & Analytics Dashboard

A full-stack CRUD dashboard built with **React**, **Spring Boot**, and **PostgreSQL** that allows users to create, view, update, and delete datasets through a clean web interface.

---

## 🚀 Tech Stack

- Frontend: React (JavaScript)
- Backend: Spring Boot (Java)
- Database: PostgreSQL
- ORM: Spring Data JPA / Hibernate
- Build Tools: Maven, Node.js

---

## ✨ Features

- Create new datasets (name + record count)
- View all datasets in a live dashboard
- Update existing dataset entries
- Delete datasets dynamically
- Real-time UI updates after each operation
- REST API integration between frontend and backend

---

## 🧱 Architecture

Frontend (React)
↓ REST API calls
Backend (Spring Boot)
↓ JPA/Hibernate
PostgreSQL Database
